# 🛠️ Оборудование и среда разработки

## Две платы в курсе

Курс разделён на **две аппаратные платформы**:

| Плата | Модули | Зачем |
|---|---|---|
| **Nucleo-F401RE** (STM32F401RE) | **Модуль 1** (основы) | Дешевле, проще, достаточно для GPIO/RCC/EXTI/Flash/TIM. Студент не перегружен лишней периферией. |
| **STM32F429I-DISC1** (STM32F429ZI) | **Модули 2-4** (периферия, DMA, RTOS) | Onboard LCD, гироскоп, акселерометр, аудио-кодек, USB OTG — всё для сложных тем. |

> **Переход:** после Лаб-01 студент пересаживается на F429I-DISC1 и больше не возвращается к Nucleo.

---

## Плата 1: Nucleo-F401RE (Модуль 1)

### STM32F401RE — Характеристики

| Параметр | Значение |
|---|---|
| MCU | STM32F401RET6 |
| Ядро | Cortex-M4 с FPU |
| SYSCLK | 84 МГц (макс) |
| Flash | 512 КБ |
| SRAM | 96 КБ |
| Тактирование | HSE 8 МГц (от ST-Link) |

### Nucleo-F401RE — Onboard-периферия

| Компонент | Пин | Использование |
|---|---|---|
| **LED (зелёный)** | PA5 | Модуль 1, GPIO, ШИМ |
| **Кнопка User** | PC13 | Модуль 1, EXTI, прерывания |
| **Кнопка Reset** | NRST | Сброс |
| **ST-Link V2-1** | SWD | Встроенный программатор + VCP (USB CDC) |
| **Arduino-разъёмы** | D0-D15, A0-A5 | Для будущих расширений (не используются в Модуле 1) |
| **Внешняя Flash M24M01** | I2C | **НЕ используется** (не поддерживается в HAL-примерах) |

### Что покрывается на Nucleo

| Тема | Компонент |
|---|---|
| Архитектура Cortex-M | LED PA5, кнопка PC13 |
| RCC / тактирование | SYSCLK 84 МГц, PLL |
| GPIO | LED, кнопка, Arduino-пины |
| Flash | Внутренняя Flash (512 КБ, секторы) |
| HAL / LL | Сравнение на LED |
| NVIC + EXTI | Кнопка PC13 → EXTI13 |
| Базовые таймеры | TIM2 → ШИМ LED |

## Плата 2: STM32F429I-DISC1 (Модули 2-4)

### STM32F429ZI — Характеристики

| Параметр | Значение |
|---|---|
| MCU | STM32F429ZIT6 |
| Ядро | Cortex-M4 с FPU |
| SYSCLK | 180 МГц (макс) |
| Flash | 2 МБ |
| SRAM | 256 КБ + 64 КБ CCM |
| Тактирование | HSE 8 МГц |
| LCD-контроллер | LTDC (onboard) |

### STM32F429I-DISC1 — Onboard-периферия

| Компонент | Подключение | Использование в курсе |
|---|---|---|
| **LED зелёный** | PG13 | Модули 2-4, GPIO, ШИМ |
| **LED оранжевый** | PG14 | Модули 2-4, GPIO, EXTI |
| **LED красный** | PG15 | Модули 2-4 |
| **LED синий** | PD4 | Модули 2-4 |
| **LCD TFT 2.4"** | FSMC + LTDC, 240×320 (RGB565) | Модули 2-4, замена внешнего дисплея |
| **Тачсклон (резистивный)** | SPI2 + GPIO | Модуль 2, ввод данных |
| **Гироскоп L3GD20** | SPI1, 3 оси | Модуль 2, замена внешнего датчика движения |
| **MEMS акселерометр LIS3DSH** | SPI1, 3 оси | Модуль 3, замена внешнего датчика |
| **Аудио-ЦАП (CS43L22)** | I2S3 + I2C1 | Модуль 2, I2S/DMA + I2C |
| **Кнопка User** | PA0 (EXTI0) | Модули 2-4, EXTI, прерывания |
| **USB OTG FS** | PA11/PA12 | Проекты, USB CDC |
| **ST-Link V2-B** | SWD | Встроенный программатор |
| **Ethernet (DP83848)** | RMII | За рамками курса |

### Что НЕ требуется закупать

Все занятия выполняются **только на встроенной периферии**. Дополнительные модули **не нужны**:

| Вместо внешнего модуля | Используем на F429I-DISC1 |
|---|---|
| BMP280 (температура/давление) | L3GD20 (гироскоп) или LIS3DSH (акселерометр) |
| W25Q64 (внешняя Flash) | Внутренняя Flash (2 МБ, сектора 0-11) |
| OLED дисплей | LCD TFT 2.4" (240×320) |
| Кнопки / энкодер | User button (PA0) + тачсклон |
| Внешний ADC | Встроенный ADC + внутренний температурный датчик |
| EEPROM | Внутренняя Flash (эмуляция EEPROM) |

## Сравнение плат

| Компонент | Nucleo-F401RE | F429I-DISC1 |
|---|---|---|
| LED | PA5 (1 шт) | PG13, PG14, PG15, PD4 (4 шт) |
| Кнопка | PC13 | PA0 |
| LCD | ❌ | ✅ 2.4" TFT |
| SPI-сенсоры | ❌ | ✅ L3GD20 + LIS3DSH |
| I2C-устройства | ❌ | ✅ CS43L22 |
| USB CDC | ✅ (ST-Link VCP) | ✅ (USB OTG FS) |
| Аудио (I2S) | ❌ | ✅ CS43L22 |
| Ethernet | ❌ | ✅ (не используется) |
| Arduino-разъёмы | ✅ | ❌ |

## Переход между платами

| Этап | Плата | Причина |
|---|---|---|
| Лекции 01-06, Практики 01-03, Лаб-01 | **Nucleo-F401RE** | Достаточно LED + кнопка. Студент фокусируется на концепциях, а не на сложной периферии. |
| Лекции 07-18, Практики 04-09, Лаб-02-04 | **F429I-DISC1** | Нужны LCD, сенсоры, DMA, USB — всё onboard. |
