# 🛠️ Оборудование и среда разработки

## Аппаратная часть
- **Плата:** STM32F401RET / STM32F429I-DISC1 
- **Программатор:** ST-Link V2 
- **Отладка:** SWD интерфейс, 4 контакта (3.3V, GND, SWDIO, SWCLK)

## Программная часть
- **IDE:** STM32CubeIDE / VS Code + PlatformIO
- **Компилятор:** arm-none-eabi-gcc
- **Отладчик:** OpenOCD / pyOCD
