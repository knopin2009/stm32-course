# Конвейер Данных

## 📋 Описание
Система сбора данных с нескольких датчиков с обработкой через DMA и передачей по USART.

## 🎯 Цели
- Мультиплексирование ADC с DMA
- Настройка USART для передачи данных
- Фильтрация данных (скользящее среднее, медианный фильтр)
- Протокол передачи данных

## 🛠️ Сборка
```bash
cd 05-Проекты/Проект-Конвейер-Данных
pio run -t upload
```

## 🚀 Запуск
1. Подключите датчики к каналам ADC
2. Подключите USB-UTL адаптер
3. Откройте Serial Monitor

## 📁 Структура
- `src/` — исходный код
- `include/` — заголовочные файлы
- `platformio.ini` — конфигурация
