# IoT Узел

## 📋 Описание
IoT-узел с беспроводной связью, датчиками и low-power режимами для автономной работы.

## 🎯 Цели
- Интеграция с беспроводным модулем (ESP-01 / LoRa)
- Low-power режимы (Sleep, Stop, Standby)
- Пробуждение по RTC и внешним прерываниям
- Протокол MQTT/CoAP для передачи данных

## 🛠️ Сборка
```bash
cd 05-Проекты/Проект-IoT-Узел
pio run -t upload
```

## 🚀 Запуск
1. Подключите беспроводной модуль по USART
2. Настройте параметры сети
3. Прошейте и наблюдайте передачу данных

## 📁 Структура
- `src/` — исходный код
- `include/` — заголовочные файлы
- `platformio.ini` — конфигурация
