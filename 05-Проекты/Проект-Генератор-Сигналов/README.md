# Генератор Сигналов

## 📋 Описание
Генератор сигналов (синус, меандр, пила) на базе STM32 с использованием DAC и DMA.

## 🎯 Цели
- Настроить DAC для генерации аналоговых сигналов
- Использовать DMA для автоматической передачи данных
- Создать таблицу значений для разных форм сигналов
- Реализовать регулировку частоты

## 🛠️ Сборка
```bash
cd 05-Проекты/Проект-Генератор-Сигналов
pio run -t upload
```

## 🚀 Запуск
1. Подключите осциллограф к выходу DAC
2. Прошейте микроконтроллер
3. Наблюдайте сигнал

## 📁 Структура
- `src/` — исходный код
- `include/` — заголовочные файлы
- `platformio.ini` — конфигурация
