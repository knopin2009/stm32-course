# Интерактивный Контроллер

## 📋 Описание
Интерактивный контроллер на базе STM32 с использованием кнопок, дисплея и обратной связи.

## 🎯 Цели
- Изучить работу с GPIO (вход/выход)
- Освоить прерывания EXTI
- Реализовать debounce кнопок
- Подключить LCD/OLED дисплей

## 🛠️ Сборка
```bash
cd 05-Проекты/Проект-Интерактивный-Контроллер
pio run -t upload
```

## 🚀 Запуск
1. Подключите ST-Link к плате
2. Прошейте микроконтроллер
3. Нажмите кнопку RESET

## 📁 Структура
- `src/` — исходный код
- `include/` — заголовочные файлы
- `platformio.ini` — конфигурация
